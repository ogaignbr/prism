# PRISM オンライン購入機能 セットアップ手順

このサイトに「カート・カード決済（Stripe）・銀行振込・注文管理（Googleスプレッドシート）」を組み込みます。
コード側の実装は完了済みで、あとは **(1) 依頼主にアカウント作成と情報をもらう → (2) 各サービスを設定 → (3) テスト → 本番公開** の流れです。

---

## パート2：設定担当者向け 作業手順

### 0. コード側で差し替える箇所（情報が揃ったら）
- `index.html` … `data-api-key="YOUR_SNIPCART_PUBLIC_API_KEY"` を **Snipcart の公開APIキー**に差し替え。
  - 商品の `data-item-url` は公開URL `https://ogaignbr.github.io/prism/` を指定済み（独自ドメインに変える場合は全16商品分を更新）。
- `legal.html` … `[[ ]]` を依頼主からの情報で差し替え（特商法・プライバシー・返品）。
- `apps-script/order-sync.gs` … `SHEET_ID` と `SECRET_TOKEN` を設定。

### 1. Snipcart ダッシュボード設定
1. **通貨を JPY** に設定（Store Configuration）。
2. **Stripe を接続**（Payment Gateway → Stripe を連携）。
3. **Deferred Payments（後払い）を有効化** → 表示名を「銀行振込」に。
   「Only allow deferred payments」は**オフ**（カードと併用）。
4. **送料・税**を設定（依頼主の送料ルールどおり）。
5. **許可ドメイン**に `ogaignbr.github.io` を登録（商品価格の検証クロール用）。
6. 注文確認メールのテンプレートに**振込先口座・振込期限**を記載（銀行振込を選んだお客様への案内）。

### 2. 注文 → Google スプレッドシート連携
1. スプレッドシートを新規作成し、ID を `apps-script/order-sync.gs` の `SHEET_ID` に設定。
2. `order-sync.gs` の内容を「拡張機能 → Apps Script」に貼り付け、`SECRET_TOKEN` を任意の文字列に変更。
3. 「デプロイ → ウェブアプリ（実行：自分／アクセス：全員）」で URL を取得。
4. Snipcart の **Settings → Webhooks** に `（その URL）?token=（SECRET_TOKEN）` を登録。
5. 対象イベント：`order.completed`（必要に応じ `order.status.changed` 等）。
   - ※ファイル冒頭のコメントに詳細手順あり。Apps Script の `testAppend` を手動実行すると動作確認できる。

### 3. テスト（Snipcart テストモード）
1. `index.html` にテスト用公開キーを入れてデプロイ。
2. 「購入する」→ カート追加 → 合計表示を確認。
3. **カード**：テストカード `4242 4242 4242 4242`（有効期限は未来、CVC任意）で決済完走 → Snipcart に注文が出るか。
4. **銀行振込（後払い）**：注文が「入金待ち」で確定し、確認メールに振込先が出るか。
5. いずれの注文も **スプレッドシートに行が追加**されるか。

### 4. 本番公開
1. 依頼主の Stripe 本人確認が完了していることを確認。
2. `legal.html` の特商法表記等がすべて記入済みであることを確認（**未記入では販売不可**）。
3. Snipcart・Stripe を**本番モード**に切替、`index.html` の APIキーを**本番公開キー**へ。
4. デプロイして最終確認。

## 費用の目安（依頼主負担・申込時に最新を要確認）
- Snipcart：テスト無料／本番は売上の約2%、または月商が一定額未満なら月額最低料金（$20前後）。
- Stripe（カード手数料）：国内カード 約3.6%／件。銀行振込分は Stripe 手数料なし。
- Google スプレッドシート／Apps Script：無料。

## 注意点
- **特定商取引法に基づく表記は法的に必須**。`legal.html` の記入が終わるまで本番公開しない。
- 価格は**税込（総額表示）**で統一（日本の表示義務）。
- 本番カード決済には **Stripe 本人確認**が前提（審査に数日）。
